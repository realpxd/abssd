# Images Directory

## Logo Image

Place your organization logo here with the filename: `logo.png` or `logo.jpg`

The logo should be the circular logo with:
- Orange outer ring
- Light green inner circle
- Gandhi glasses symbol in the center
- Hindi text "अखिल भारतीय स्वच्छता सेवा दल ट्रस्ट" around the ring

**Recommended specifications:**
- Format: PNG (preferred) or JPG
- Shape: Circular/square (will be displayed as circle)
- Recommended size: 512x512px or larger
- File name: `logo.png` (or `logo.jpg`)

The component will automatically use this image. If PNG is not found, it will try JPG.

## Volunteer Showcase Image

Place your volunteer image here with the filename: `volunteers.png`

The image should show volunteers in orange aprons and caps, similar to the promotional poster.

**Recommended specifications:**
- Format: PNG or JPG
- Aspect ratio: 16:10 (or similar landscape format)
- Recommended size: 1200x750px or larger
- File name: `volunteers.png`

The component will automatically use this image. If the image is not found, it will fall back to a placeholder.
